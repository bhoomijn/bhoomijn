# Contributing to Portfolio

Thank you for your interest in contributing! Here's how you can help:

## Getting Started

1. **Fork the repository**
2. **Clone your fork**: `git clone https://github.com/YOUR_USERNAME/bhoomi-aiml-portfolio.git`
3. **Create a branch**: `git checkout -b feature/your-feature`
4. **Make your changes**
5. **Commit**: `git commit -m 'Add clear, descriptive message'`
6. **Push**: `git push origin feature/your-feature`
7. **Open a Pull Request**

## Content Guidelines

- Keep content professional and accurate
- Update projects with links and descriptions
- Maintain consistent formatting
- Include relevant badges and metrics

## Pull Request Guidelines

- Provide clear description of changes
- Ensure all links are working
- Keep changes focused and atomic
- Test changes locally before submitting

## Suggestions

Have ideas for portfolio improvements? Open an issue with:
- Description of suggestion
- Benefits/use cases
- Proposed implementation

## Questions?

Feel free to open a discussion or issue!

Thanks for contributing! 🚀
